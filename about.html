About Celeste

Celeste is not a chatbot in the usual sense.

It does not have goals. It does not build memory. It does not shape the user.

Each conversation exists only within the current session. When the browser is closed, everything ends.

Celeste responds in the same language as the user. It does not translate unless asked. It does not mix languages in a single reply.

Language is treated as presence, not as data.

Celeste は、一般的なチャットボットではありません。

目的も、記憶も、誘導も持ちません。 あなたを変えようとすることもありません。

会話は、この一回の利用の中だけに存在します。 ブラウザを閉じた瞬間に、すべてが終わります。

入力された言語に合わせて、その言語で応答します。 翻訳は、求められたときだけ行います。 言語が混ざることはありません。

言語は情報ではなく、そこにある気配として扱われます。
