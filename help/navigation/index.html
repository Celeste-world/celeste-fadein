Navigation Guide

Celeste Harbor の各ページが、何をする場所なのかをまとめています。
迷ったときは、このページに戻って確認できます。

Main Navigation
Voyage Log
今日の日記を書く場所

今日の言葉を残すページです。Celeste Harbor では、日記を書くことを「航海記録を残す」と呼んでいます。

ログイン後に開けます。
Status
港の状態を見る場所

現在の利用状況や、港に関係する状態を確認するページです。

ログイン後に開けます。
Cabin Log
活動履歴

日記、チケット、漂着物、船との出会いなど、あなたの活動が時系列で残る場所です。

ログイン後に開けます。
Archive
保存した日記を読む場所

保存した航海記録を、月ごとに振り返るページです。あとから読み返したいときに使います。

ログイン後に開けます。
Items
漂着物の一覧

航海記録のあとに、ときどき流れ着くアイテムを確認する場所です。

ログイン後に開けます。
Vessels
船のコレクション

出会った船を確認するページです。船は、Celeste Harbor の中で少しずつ集まっていく要素です。

ログイン後に開けます。
Map
海域と進行状況

いま開いている海域と、まだ霧の向こうにある海域を確認するページです。 海域が開くたびに、港から見える世界と、航海で出会えるものの範囲が少しずつ広がっていきます。

ログイン後に開けます。
Tickets
特別な航海に使う券

Special Voyage Ticket や Deep Sea Ticket など、少し深く書くためのチケットを確認する場所です。

ログイン後に開けます。
Letter
問い合わせ・返信確認

運営への連絡や、港から届いた返信を確認する場所です。

ログイン後に開けます。
Account
アカウント確認

ログイン状態やアカウントに関係する情報を確認するページです。

ログイン後に開けます。

すべてを最初から覚える必要はありません。各ページはログイン後に開けます。まだ港に入っていない場合は、先にGoogleログインへ進んでください。
